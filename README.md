# 💰 Finanzas Familia V&V

Dashboard de salud financiera y comité familiar — conectado a Google Sheets.

## 🚀 Live Dashboard
👉 [Abrir Dashboard](https://diegovalenzuelaca-hub.github.io/finanzas-familia-vv/)

## Archivos
- `index.html` — Dashboard HTML (GitHub Pages)
- `streamlit_app.py` — Wrapper para Streamlit Cloud

## Cómo conectar tus datos
1. Abre el dashboard
2. Ingresa el ID de tu Google Spreadsheet
3. El Sheet debe estar compartido como "Cualquiera con el enlace puede ver"
